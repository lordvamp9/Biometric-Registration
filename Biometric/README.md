# BioLogistics Registration

Secure driver registration system with biometric liveness verification.

## Overview
Web application for logistics identity verification using webcam-based face detection.

## Features
- Split-screen professional UI
- Real-time face detection
- Liveness challenge (Head turn)
- Automated verification flow

## Tech Stack
- HTML, CSS, JavaScript
- face-api.js

## How to Run
1. Serve the project folder using a local web server.
2. Open index.html through the server URL (e.g., http://localhost:8000).
3. Allow camera access when prompted.
